Mira

To Run a Module
Export PYTHONPATH
```bash
export PYTHONPATH=/path/to/mira_sdk
```

## Overview

Mira is an SDK for running and managing Automation efficiently.

## Installation

Clone the repository:

```bash
git clone https://github.com/Marvell-Central/mira_sdk.git
cd mira_sdk
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Running a Test Case

1. Set the `PYTHONPATH` as shown above.
2. Run your test case:

```bash
python tests/test_example.py
```
